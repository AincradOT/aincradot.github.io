# Contributing to the Aincrad Wiki

This wiki is an open project that welcomes contributions from all players. If you'd like to help improve the wiki:

## How to Contribute

1. Visit our [GitHub Repository](https://github.com/aincradot/aincradot.github.io)
2. Create a fork of the repository
3. Make your changes and submit a pull request
4. Wiki maintainers will review and merge your contributions

## What You Can Contribute

* **New guides** - Help document game features, quests, or mechanics
* **Updates** - Keep existing information accurate and up-to-date
* **Corrections** - Fix errors or clarify confusing information
* **Formatting** - Improve readability and organization
* **Translations** - Help make the wiki accessible to more players

## Guidelines

* Keep information accurate and verified
* Follow the existing formatting style
* Write clearly and concisely
* Test all links before submitting

!!! tip "Need Help?"
If you're unsure about how to contribute or have questions, feel free to open an issue on GitHub or ask in our Discord channel.
!!!
